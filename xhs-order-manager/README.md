# 🌹 商单管家 - 小红书博主商单管理小程序

专为小红书接广博主打造的商单管理工具，帮你高效管理商单、报价和品牌关系。

## 📦 功能模块

### 1. 🏠 首页看板
- 本月商单数、收入、返点概览
- 各状态商单快速入口（待确认/已确认/拍摄中/已发布）
- 近7天截稿预警
- 最近商单列表

### 2. 📋 商单管理
- **商单列表**：支持按状态筛选、关键词搜索
- **新建/编辑商单**：
  - 品牌名称（可从品牌库选择，自动带入联系方式）
  - 内容类型（图文/视频/直播/测评等）
  - 报价金额 & 返点设置
  - 截稿日期
  - 品牌需求 & 备注
  - 联系人信息
- **商单详情**：
  - 可视化进度条（待确认→已确认→拍摄中→已发布→已结算）
  - 一键修改状态
  - 联系方式一键复制/拨打

### 3. 💰 报价计算器
- 基于 **粉丝数 × 互动率 × 内容类型** 智能估算报价
- 给出建议报价和合理区间（低20% ~ 高20%）
- 支持保存报价模板，下次一键套用
- 一键复制报价单文案

### 4. 🎁 返点管理
- 记录每笔商单的返点情况
- 支持百分比返点/固定金额/产品置换
- 返点状态追踪（待收取/已收取/逾期）
- 总返点统计（应收/已收/待收）
- 一键标记为已收取

### 5. 🏢 品牌管理
- 品牌信息档案（名称、行业、官网）
- 联系人信息（姓名、职位、微信/电话/邮箱）
- 合作政策记录（返点政策、常规返点率）
- 一键拨打/复制联系方式
- 展示该品牌历史合作商单

### 6. 📊 数据统计
- 支持本月/本季度/今年/全部 4种周期
- 有效商单数、总报价、已收返点、结算率、平均报价
- 近6个月收入趋势柱状图
- 内容类型分布横条图
- 品牌收入排行榜（TOP 5）

## 🚀 快速开始

1. **下载微信开发者工具**：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

2. **导入项目**：
   - 打开微信开发者工具
   - 选择「导入项目」
   - 项目目录选择 `xhs-order-manager/`
   - AppID 填入你自己的小程序 AppID（测试用可用测试号）

3. **运行**：点击「预览」或「真机调试」即可使用

## 📂 项目结构

```
xhs-order-manager/
├── app.js              # 全局逻辑 & 数据初始化
├── app.json            # 全局配置（页面路由、TabBar）
├── app.wxss            # 全局样式
├── utils/
│   ├── storage.js      # 数据存储工具（本地缓存CRUD）
│   └── util.js         # 通用工具函数（格式化、状态配置等）
├── pages/
│   ├── index/          # 首页看板
│   ├── orders/
│   │   ├── list/       # 商单列表
│   │   ├── add/        # 新建/编辑商单
│   │   └── detail/     # 商单详情
│   ├── quote/          # 报价计算器
│   ├── rebate/         # 返点管理
│   ├── brands/
│   │   ├── list/       # 品牌列表
│   │   └── detail/     # 品牌详情/编辑
│   └── stats/          # 数据统计
└── images/             # 图标资源（TabBar 图标需自行添加）
```

## ⚙️ 数据说明

所有数据存储在微信小程序本地缓存（`wx.setStorageSync`），不需要服务端。

| Key | 说明 |
|-----|------|
| `orders` | 商单列表 |
| `brands` | 品牌联系方式 |
| `rebates` | 返点记录 |
| `quoteTemplates` | 报价模板 |
